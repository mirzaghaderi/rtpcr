# Sample qPCR data: amplification efficiency

A sample qPCR dataset for demonstrating efficiency calculation.

## Usage

``` r
data_efficiency
```

## Format

A data frame with 21 observations and 4 variables:

- dilutions:

  Dilution factor

- C2H2.26:

  Target gene 1

- C2H2.01:

  Target gene 2

- GAPDH:

  Reference gene
