# Sample data (with technical replicates)

A sample data for calculating biological replicated. Each line belongs
to a separate individual (non-repeated measure experiment).

## Usage

``` r
Lee_etal2020qPCR
```

## Format

A data frame with 72 observations and 8 variables:

- factor1:

  experimental factor

- DS:

  DS

- biolRep:

  biological replicate

- techRep:

  technical replicates

- E_APOE:

  Amplification efficiency of APOE gene

- Ct_APOE:

  Ct of APOE gene

- E_GAPDH:

  Amplification efficiency of GAPDH gene

- Ct_GAPDH:

  Ct of GAPDH gene

## Source

Lee et al, (2020) \<doi:10.12688/f1000research.23580.2\>
