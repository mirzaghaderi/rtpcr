# Sample data (with technical replicates)

A sample data for calculating biological replicated. Each line belongs
to a separate individual (non-repeated measure experiment).

## Usage

``` r
data_withTechRep
```

## Format

A data frame with 18 observations and 9 variables:

- Condition:

  experimental factor

- biolrep:

  biological replicate

- techrep:

  technical replicates

- target:

  Amplification efficiency of target gene

- Ct_target:

  Ct of target gene

- ref:

  Amplification efficiency of reference gene

- Ct_ref:

  Ct of reference gene
